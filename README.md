# CARROS TURBO 🏎️💨

Corredor sin fin *top-down* jugable en el **móvil**, hecho con HTML5 Canvas. Esquiva el tráfico, recoge monedas 🪙 y llega lo más lejos posible mientras la velocidad sube sola. Es una **PWA instalable** 📲.

## 🌐 Juégalo en vivo
# 👉 https://samuelcatalanz123.github.io/carros-turbo/

## 📲 Instalar en el teléfono
- **Android (Chrome):** menú ⋮ → "Instalar aplicación" (o el botón *📲 Instalar app*).
- **iPhone (Safari):** Compartir ↑ → "Añadir a pantalla de inicio".

## 🎮 Cómo jugar
- **Arrastra el dedo** para mover el carro (o usa las flechas ← → en la computadora).
- **Esquiva** los carros que vienen y **agarra monedas**.
- ¡Chocas = fin! Se guarda tu **récord** de metros y tus monedas totales.
- Elige el **color de tu carro** en el menú.

## 🧰 Tecnología
HTML5 Canvas + JavaScript puro · PWA (manifiesto + service worker) · GitHub Pages.
